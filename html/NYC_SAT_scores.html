Study of the SAT scores for NYC schools¶

In this short project we study the average SAT scores of schools in New York City (NYC) using data from the 2010s downloaded from the NYC Open Data website. We aim at finding correlations between SAT test results and a school's geographical or social situation.

Load the data files¶

We first import the packages we will use: Pandas for woking with datasets, Numpy to perform calculations efficiently, re for working with regular expressions, Pyplot to represent results graphically, and Basemap and Geopandas to draw maps.

In [3]:
def pad_csd(num):
    '''
    pad a number with a 0 if it has only one digit and return its string 
    representation; num is a positive integer 
    '''
    if num > 9:
        return str(num)
    else:
        return '0' + str(num)
    
data['class_size']['DBN'] = data['class_size']['CSD'].apply(pad_csd) + data['class_size']['SCHOOL CODE']

Data cleaning¶

We first convert the columns containing string representation of numbers to numeric values, add a column sat_score to sat_results containing the total averaged SAT score, and add columns lat and lon to hs_directory contzining, respectively, the latitude and longitude of the school locations.

In [4]:
# columns from the sat_results dataframe whose entries should be converted
cols = ['SAT Math Avg. Score', 'SAT Critical Reading Avg. Score', 'SAT Writing Avg. Score']

# change the values to numeric
# "errors='coerce'" ensures that entries not properly parsed are set to NaN
for col in cols: 
    data['sat_results'][col] = pd.to_numeric(data['sat_results'][col], errors='coerce')
    
# compute the total SAT score and store it in the sat_score column of the sat_results dataframe
data['sat_results']['sat_score'] = data['sat_results'][cols[0]] + data['sat_results'][cols[1]] + data['sat_results'][cols[2]]

# columns from the ap_2010 dataframe whose entries should be converted
cols = ['AP Test Takers ', 'Total Exams Taken', 'Number of Exams with scores 3 4 or 5']

# change the values to numeric
# "errors='coerce'" ensures that entries not properly parsed are set to NaN
for col in cols:
    data['ap_2010'][col] = pd.to_numeric(data['ap_2010'][col], errors='coerce')

# function to find the latitute from strings in the 'Location 1' columns of the 'hs_directory' dataset
def find_lat(location):
    try:
        return float(re.search('\((.+), .+\)', location)[1])
    except TypeError:
        return np.nan
    
# function to find the longitute from strings in the 'Location 1' columns of the 'hs_directory' dataset
def find_lon(location):
    try:
        return float(re.search('\(.+, (.+)\)', location)[1])
    except TypeError:
        return np.nan

# create the columns lat and lon in the hs_directory dataframe, containing 
# respectively the latitude ad longitude of the schools
data['hs_directory']['lat'] = data['hs_directory']['Location 1'].apply(find_lat)
data['hs_directory']['lon'] = data['hs_directory']['Location 1'].apply(find_lon)

We then condense the datasets, keeping only the relevent information.

In [5]:
# keep only class size data related to grades 9 to 12 for the 'GEN ED' program 
class_size = data['class_size'][
    (data['class_size']['GRADE '] == '09-12')
    & (data['class_size']['PROGRAM TYPE'] == 'GEN ED')
]

# keep only one row by DBN, averaging over entries with the same DBN
class_size = class_size.groupby('DBN').agg(np.mean)
class_size.reset_index(inplace=True)

# update the class_size dataframe in the data disctionary
data['class_size'] = class_size

# keep only demographic data for the year 2011-2012
data['demographics'] = data['demographics'][data['demographics']['schoolyear'] == 20112012]

# keeping only graduation data correponding to the total 2006 cohort
data['graduation'] = data['graduation'][
    (data['graduation']['Cohort'] == '2006')
    & (data['graduation']['Demographic'] == 'Total Cohort')
]

SAT score versus AP exams¶

We now want to see if, and to what extent, the percentage of students taking AP exams is correlated with a school's averaged SAT score. To this end, we first add a column ap_per to combined giving this percentage. We then show a scatter plot of the averaged SAT score versus this new column, and compute their correlation.

In [18]:

plt.show()
print('Correlation coefficient: {}'.format(combined[['sat_score','ap_per']].corr()['sat_score'][1]))

Correlation coefficient: 0.057170813907669635

The correlation coefficient is positive, but quite small. From the scatter plot, it seems that some, but not all, schools with a high percentage of AP test takers have higher SAT averages. The data seems to point to two distinct categories of schools: one whose averaged SAT is strongly correlated with the percentage of students taking AP tests and one for whom these numbers are uncorrelated.

The first group of schools is made of those with a high average SAT score. As mentioned above, they tend to be old, well-known schools. The second group contains, notably, all schools with more than 70% of AP test takers. Let us print their names:

Conclusions¶

The two main conclusions from this brief study of the SAT scores of NYC secondary schools are

  • the positive correlations between the average SAT scores and the perceived safety, respect, and engagement scores,
  • the fact that schools with an important fraction of pupils from minorities can have lower SAT scores despite a larger fraction of pupils taking AP tests.

This second point may indicate that programs helping recent immigrants or other minorities are needed to ensure all pupils are given the same opportunities to succeed.
